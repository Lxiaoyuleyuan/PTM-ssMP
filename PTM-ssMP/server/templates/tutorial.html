User Guide

Step1: Input the protein sequence with verified phosphorylation site

Input format: You can either load a protein list from a file or paste a protein list to the text area. PTM-ssMP was designed to accept the data in FASTA format. In order not to inconvenience anyone, please ensure that known type and position of PTM sites is listed behind the protein name, separating them using ‘|’. In order to effectively utilize the known post-translational modification information, each known PTM type is separated by ‘;’, and the PTM different site position information is separated by ‘,’, just as shown in below. In addition, for convenience, we define abbreviations for each PTM name and the name table is shown in Table1. Any problem, please contact us.

Step2: Select the PTM model you concern

You can select one or multiple PTM once. We classify all the PTM onto three groups: one including S/T residues, K residues and the other including Y residues.

Step3: Submit the form to the backend server pushing the ‘Submit form’ button

Here, you can use the ‘Reset all’ button if you want to return to the default page. And we offer three selectable thresholds of 90%, 95%, 99% respectively.

Step 4. Results are shown in the page after a few minute

This is a 'overview' list including the result of a threshold of 90%. It contains Job ID, result table, Parameters and so on. The corresponding results will be shown on the panel like the following figure. It contains Protein ID, Locations, PTM types, Residues, local sequence and Score. And predictive results can be exported as JSOPN, XML, CSV, TXT, SQL, MS-Excel format files. We also generate a Job ID for each prediction, and users can search for results in the search box via the Job ID.
